# Traffix Simulator made in C++ SFML
It is a fully working Traffic simulator game made in C++ and SFML where we can also control our personal vehicle and travel to differnet places <br>
IDE used Code Blocks <br>
All the code is written insie "main.cpp" file in the main directory
## How to run:
To run it, go inside bin>Release and run "text.exe" File to run the game

## Concepts I used:
Each signal where 4 roads meet work as a node <br>
There are several nodes in this map <br>
Roads which are coming off map are called ghost nodes<br>
The actual roads are called edges<br>
Our personal vehicle will be parked in police station by default<br>
We can click on any property on map and our personal vehicle will find the <br>
Shortest path to that property using BFS (Beath First Search) and will travel <br>
to that property following all the traffic rules i.e running on left lane<br>
and following traffic rules and not colliding in any other vehicle.<br>
Ambulance don't followt he signals because its emergency vehicle
